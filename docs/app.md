# App python files

This is where the main logic is executed for the different operations in the project:

1. [app.py](#apppy): Web-scraping and directory cleanup (duplicate rate card pdfs)
    - Data extracted here located in *'database/bronze'* directory
2. [app2.py](#app2py): Rate card table extraction, cleaning, and transformation 
    - Data produced here stored in silver and gold folders

3. [app3.py](#app3py): Plot SFIA level day rate distributions and corresponding monte carlos
    - Plots output to '*gold/plots*' directory
